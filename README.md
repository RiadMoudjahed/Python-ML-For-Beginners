# Python ML For Beginners

A collection of beginner-friendly machine learning projects in Python. This repository serves as a hands-on introduction to essential ML concepts using real-world datasets.

## 📁 Projects

### 1. MNIST Handwritten Digit Classification (`Ai CNN.py`)
A Convolutional Neural Network (CNN) built with PyTorch to classify handwritten digits from the MNIST dataset.

**Key concepts:**
- Loading and preprocessing image data
- Building a CNN with convolutional and pooling layers
- Training loop and model evaluation
- Confusion matrix visualization

### 2. Housing Price Prediction (`housing_price_prediction.py`)
Predict median house values using linear regression and random forest regressors.

**Key concepts:**
- Data cleaning and handling missing values
- One-hot encoding categorical variables
- Train-test split
- Linear Regression vs Random Forest Regressor
- Model evaluation (R², MSE, RMSE)

### 3. Email Spam Classifier (`email-spam-classifier.py`)
Classify emails as spam or not spam using logistic regression and TF-IDF vectorization.

**Key concepts:**
- Text preprocessing and cleaning
- TF-IDF (Term Frequency-Inverse Document Frequency) vectorization
- Logistic regression for binary classification
- Classification report and confusion matrix

## 📊 Datasets Used

- **MNIST**: Built-in from `torchvision.datasets`
- **Housing Dataset**: Requires `housing.csv` (California housing dataset)
- **Email Dataset**: Requires `emails.csv` (spam classification dataset)

## 📈 What You'll Learn
- Data preprocessing (handling missing values, encoding categorical data)
- Model building (CNNs, Linear Regression, Random Forest, Logistic Regression)
- Model evaluation (accuracy, confusion matrix, R², MSE, RMSE)
- Text processing (TF-IDF, text cleaning)
